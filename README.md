Cybersecurity Portfolio

This repository showcases hands-on cybersecurity projects focused on threat detection, incident analysis, and network security.

Projects
Phishing Triage Tool

Developed a Python-based email analysis tool that evaluates SPF, DKIM, and DMARC authentication results, header alignment, and suspicious links to identify phishing attempts. The tool successfully detected high-risk phishing emails while maintaining zero false positives during testing.

ARP Spoofing Detection

Built a network analysis solution using Scapy that detects ARP spoofing attacks by monitoring IP-to-MAC relationships and identifying conflicting gateway claims within packet captures. The project includes a synthetic attack dataset and analyst investigation report.
